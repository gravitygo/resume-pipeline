# Resume Templates

## Available Themes

### Modern
- Clean, contemporary design
- Blue accent colors
- Sans-serif typography
- Clear section borders

### Classic
- Traditional, timeless look
- Black and white color scheme
- Serif typography
- Centered name header

### Compact
- Space-efficient layout
- Smaller font sizes
- Minimal margins
- Ideal for lengthy resumes

## Template Structure

All templates use the same HTML structure with different CSS styling:

1. Header with candidate name (h1)
2. Section headers (h2)
3. Job entries (h3)
4. Bullet points (ul/li)
5. Paragraphs for summary (p)

## Styling Approach

- Styles defined in JSON configuration
- Converted to CSS at render time
- Injected into HTML template
- Print-optimized for PDF generation
