# Resume Generation Pipeline

## Purpose

This system transforms structured career data into professional, ATS-friendly PDF resumes using AI.

## Pipeline Flow

1. Accept structured input data (name, experiences, skills, certifications)
2. Generate Markdown resume via LLM
3. Convert Markdown to HTML
4. Apply theme styling
5. Render PDF using headless browser

## Output Characteristics

- Clean, professional formatting
- ATS-compatible (selectable text, proper structure)
- Consistent across runs
- Multiple theme support
